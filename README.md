## DESCRIPTION

The Color Changer is a simple web application .. It demonstrates DOM manipulation by allowing users to change the background color of a box to a randomly generated color with the click of a button.

## USAGE

Open the project in Visual Studio Code.
Open index.html in your browser or run it using the Live Server extension.
Click the Change Color button to generate a new random color.
